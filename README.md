# UE to ARKit Shape Key Converter
A Blender plugin for converting shape keys/blendshapes from the Unified Expressions standard to the ARKit standard.

## Installation
- Download the python file from the releases section on the right (or the source above)
- In Blender, go to Edit > Preferences > Add-ons and click "Install from Disk" from the drop-down at the top right
- Select the downloaded file
- Check the box to enable the add-on, if not already checked

## Use
- Select the mesh containing the face tracking shape keys (typically "Body" or "Head")
- Click on the Object tab, then click "Convert UE to ARKit"
- If necessary, go to Blender's info window to see which shape keys are missing
